Ao criar um estilo qualquer, qual o melhor padrão para marcar o nosso conteúdo, ao criar uma classe?


Ser específico e genérico ao mesmo tempo


Alternativa correta! Um balanço é a melhor alternativa. Nem tão específica, para poder repetir as classes, e nem tão genérica, para poder não precisar combinar classes.

Dica: utilize class para estilo e id para comportamento
